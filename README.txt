GENERAL NOTES:
This ROS Package is intended to control custom-made robotic research
platforms "RL500XA/XB" <https://sites.google.com/site/ruslanmasinjila/rl_500_xa>.
***************************************************************
CURRENT STATUS:
	VERSION 1.....[COMPLETE]
	-Wireless Steering of a single platform using a Joystick connected 
	 to control PC
	-Steering of the platform using RC an controller
***************************************************************
	VERSION 2.....[COMPLETE]
	-Remote Steering of a single platform over a network
	-Remote operation of the turret that holds the sensors
	-Feedback from wheel encoders
***************************************************************
	VERSION 3.....[COMPLETE]
	-Remote steering of multiple platforms simultaneously 
	 or independently
	-Distributed Communicating ROS nodes
***************************************************************
